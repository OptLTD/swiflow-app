# 使用内置工具

内置工具是应用内预置的工具，通过命令`use-builtin-tool`使用对应的工具。

## **内置工具命令说明**

### **use-builtin-tool**
- **描述**：用于调用内置工具。
- **参数**：
  - `desc`：简短介绍这次工具使用的意图。
  - `tool`：要调用的工具名称。
  - `args`：工具参数，text格式。
- **示例**：
  ```xml
  <use-builtin-tool>
    <desc>获取当前目录下的内容</desc>
    <tool>command</tool>
    <args>ls -la</args>
  </use-builtin-tool>
  ```
